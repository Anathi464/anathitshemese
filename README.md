# Flow — AI Productivity Assistant

Flow is a modern, AI-powered productivity assistant that helps professionals automate repetitive workplace tasks. Draft polished emails, summarize meetings, plan your day, research topics, and chat with an AI co-pilot — all in one clean workspace.

![Built with Lovable](https://img.shields.io/badge/Built%20with-Lovable-8B5CF6) ![React](https://img.shields.io/badge/React-18-61DAFB) ![Vite](https://img.shields.io/badge/Vite-5-646CFF) ![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6) ![Tailwind](https://img.shields.io/badge/TailwindCSS-3-38BDF8)

---

## ✨ Features

- **📧 Smart Email Generator** — Generate professional emails with selectable tone (formal, friendly, persuasive, apologetic) and audience (client, manager, recruiter, vendor).
- **📝 Meeting Summarizer** — Turn raw notes or transcripts into structured summaries with key points, decisions, and an action-items table.
- **📅 Task Planner** — Prioritizes tasks using the Eisenhower matrix and produces a realistic daily/weekly schedule with productivity tips.
- **🔎 Research Assistant** — Generates TL;DRs, key insights, recommendations, and plain-language explanations from topics or pasted articles.
- **💬 Chat Assistant** — Conversational AI co-pilot for drafting, brainstorming, planning, and quick questions.
- **📤 Export & Copy** — One-click copy or export of any AI output.
- **⚡ Real-time Streaming** — Responses stream token-by-token for a fast, modern UX.
- **📱 Responsive UI** — Works seamlessly across mobile, tablet, and desktop.

---

## 🛠️ Tech Stack

- **Frontend:** React 18, Vite 5, TypeScript 5
- **Styling:** Tailwind CSS v3, shadcn/ui, custom design tokens
- **Backend:** Lovable Cloud (Supabase Edge Functions)
- **AI:** Lovable AI Gateway (`google/gemini-2.5-flash`)
- **Streaming:** Server-Sent Events (SSE)

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) 18+ or [Bun](https://bun.sh/)

### Installation

```bash
# Clone the repository
git clone <YOUR_REPO_URL>
cd <YOUR_PROJECT>

# Install dependencies
npm install
# or
bun install

# Start the dev server
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

### Environment Variables

The `.env` file is auto-generated by Lovable Cloud and contains:

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_PUBLISHABLE_KEY=...
VITE_SUPABASE_PROJECT_ID=...
```

The `LOVABLE_API_KEY` used by the edge function is managed automatically in the Lovable Cloud backend.
